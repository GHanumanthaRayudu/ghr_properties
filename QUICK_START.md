# ⚡ Quick Start Guide

Get your GHR Properties application running in **5 minutes**!

## 🚀 Commands to Run

```bash
# 1. Install dependencies
bundle install

# 2. Install Tailwind CSS
bundle exec rails tailwindcss:install

# 3. Setup database
bin/rails db:create db:migrate db:seed

# 4. Start the app
bin/dev
```

## 🌐 Open Your Browser

Visit: **http://localhost:3000**

## 🎉 That's It!

You now have a fully functional GHR Properties platform with beautiful UI!
